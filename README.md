# challenge3-Employee-Payroll-Tracker

## Description

The Employee Payroll Tracker is a web application that helps businesses manage employee work salary, and determine the average salary across all employees. This application is built using HTML, CSS, and JavaScript.


Features:


- Add employees by entering their first name, last name and salary.

- Display a table showing employee first name, last name and salary.

- Calculate the average salary of all employees added to the console.

- Display a Random winner employee.

- Intuitive user interface for easy interaction.
